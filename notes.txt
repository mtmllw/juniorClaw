### Recent Architectural Upgrades (JuniorClaw Autonomous Setup)

1. **Host Network Binding**: Overrode the Docker port proxy with `network_mode: "host"` so the gateway natively binds to the VPS loopback interface, permanently resolving the SSH tunnel "Connection Refused" issues.
2. **Zero-Touch Auto-Installation**: Revamped `setup.sh` to safely bypass the interactive CLI wizard via the `--non-interactive` flag. It seamlessly uses `openssl` to auto-generate and inject a secure 32-character Gateway Token directly into `.env` without human prompts.
3. **Strict Pre-flight Validation**: Injected safeguards into `setup.sh` that actively abort the boot sequence with helpful terminal alerts if key LLM or GitHub configuration parameters are missing from `.env`, guaranteeing the Docker containers never boot broken.
4. **Master Orchestrator Engine (`AGENTS.md`)**: Re-prompted the core agent rulebook into a hierarchical Master Persona. The agent now aggressively plans projects by splitting tasks into a `.agent_memory/tasks.md` message bus and rapidly assuming sub-personas (Lead Architect, Frontend UI, QA Tester) sequentially.
5. **Segmented Context Silos**: Implemented distinct persona memory files (`frontend_memory.md`, `backend_memory.md`) forcing sub-personas to strictly read their own architectural silos, preventing AI hallucination from bloated LLM context windows.
6. **Enterprise GitHub App Flow**: Installed the official `gh` CLI and `python3-jwt` into `Dockerfile`. Built `get_gh_app_token.py` to allow the agent to dynamically cryptographic-sign 1-hour Installation Access tokens using a `.pem` file, safely avoiding the reliance on static Personal Access Tokens for autonomous repocasting.
7. **Visual Proof-of-Work via Telegram**: Added `imagemagick` to the Docker stack and updated `autonomous-developer/SKILL.md`. The QA Persona is now strictly mandated to boot web apps inside the `xvfb` virtual framebuffer, take live UI screenshots of the headless code rendering, and upload them alongside raw STDOUT crash logs natively over the Telegram API.
